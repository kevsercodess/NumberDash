# NumberDash

NumberDash is a simple Java math racing game for children aged 6–10. The player answers math questions to move forward on the track and compete against the CPU. Correct answers help the player advance, while incorrect answers allow the CPU to move ahead. The first racer to reach the finish line wins.

## Features

- Welcome screen
- Main menu
- How to Play section
- Math question challenge
- Player vs CPU race
- High score tracking
- Result display after each race

## How to Run the Application

1. Open the project in NetBeans or any Java IDE.
2. Compile and run the main class:
   `NumberDash.java`
3. Follow the menu options shown in the console.
4. Select:
   - Start Race
   - How to Play
   - View High Score
   - Quit

## Game Rules

- Correct answer → Player moves forward
- Wrong answer → CPU moves forward
- First to reach the finish line wins
- The score is based on correct answers

## Role / Feature Breakdown

This project was completed individually, but multiple feature branches were used to demonstrate collaboration workflow.

- **NumberDash.java** – Menu system and main application flow
- **RaceLogic.java** – Game logic, math questions, player and CPU movement
- **ScoreBoard.java** – Score tracking, result display, and high score management

## Git Workflow

The project used GitHub branching and merging to demonstrate software development workflow.

Feature branches used:
- `feature/menu`
- `feature/race-logic`
- `feature/score-system`

Additional branches were also used for documentation and small improvements.
